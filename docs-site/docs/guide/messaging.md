# Messaging

MeetPlanner has built-in messaging so your team can communicate in the same tool where work happens — and turn conversations into tasks in one click.

**Available to:** All roles

---

## Channels Sidebar

The left panel in `/messaging` lists:

| Type | Description |
|---|---|
| **Public channels** | Visible and joinable by all team members |
| **Private channels** | Invite-only, shown with a lock icon |
| **Department channels** | Auto-scoped to your department (created by Admin) |
| **Direct Messages (DMs)** | One-on-one or small group conversations |

Unread message counts appear as badges next to each channel name.

---

## Sending a Message

1. Click a channel or DM in the sidebar.
2. Type in the message input at the bottom.
3. Press **Enter** to send. Use **Shift+Enter** for a new line.

---

## Creating a Channel

1. Click **+** next to the "Channels" heading in the sidebar.
2. Enter a channel name.
3. Choose **Public** or **Private**.
4. Select a department scope (optional).
5. Add initial members.
6. Click **Create**.

---

## Message Actions

Hover over any message to reveal the action bar on the right:

| Action | Description |
|---|---|
| **React** | Add an emoji reaction |
| **Reply** | Start or continue a threaded reply |
| **Flag as Idea** | Marks the message for follow-up |
| **Create Task** | AI extracts an action item from the message and sends it to Triage |
| **Copy** | Copies the message text to your clipboard |
| **Delete** | Removes your own message only |

---

## Chat → Task Conversion

When someone posts an action item in a channel (e.g., *"John can you update the proposal by Friday"*):

1. Hover the message → click **Create Task**.
2. The AI extracts: title, suggested assignee, and due date.
3. The task is created in **Triage** status.
4. A Manager or Admin reviews and approves it in the [Triage Queue](/docs/guide/triage).
5. The task then appears on the Task Board and is visible to the assignee.

:::tip
This is the fastest way to capture action items that come up in informal conversations.
:::

---

## Message History

Scroll up in any channel to load older messages automatically (infinite scroll). Date separators appear at the start of each day to help you orient in the history.

---

## Direct Messages

To start a DM:

1. Click **+** next to "Direct Messages" in the sidebar.
2. Search for one or more team members.
3. Click **Open**. The conversation opens immediately.

All message actions (react, reply, create task, etc.) work identically in DMs.
